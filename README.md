# ቃላት Qalat - Ethiopian Story Platform

Ethiopia's home for stories — read, write, and share in Amharic, Afaan Oromo, Tigrinya & more.

## Features

- 📖 Read stories in multiple Ethiopian languages
- ✍️ Publish your own stories with photo uploads
- 🌓 Light/Dark mode toggle
- 📱 Fully responsive design
- 💬 Comment on chapters
- ❤️ Like and track story stats
- 🎨 Colorful Ethiopian flag-inspired design

## Deploy to Render

1. Push this repo to GitHub
2. Go to [render.com](https://render.com)
3. Create a new **Static Site**
4. Connect your GitHub repo
5. Build Command: `npm install`
6. Publish Directory: `.` (current directory)
7. Click **Create Static Site**

Your site will be live at `https://your-site-name.onrender.com`

## Local Development

Open `index.html` in your browser. No build step needed!

## Tech Stack

- Pure HTML, CSS, JavaScript
- No frameworks or dependencies
- Ethiopian flag colors (Green, Yellow, Red)
- Noto Sans Ethiopic font for Amharic script
